Zero to Running

Quick Start

Up and running in under 2 minutes.

1

Pull & Run

Single container — no compose file needed.

docker run -d \
  -p 8090:8080 \
  -v mcp-data:/app/data \
  ghcr.io/your-username/mcp-explorer-x:latest
2

Open the App

Navigate to the Connections page.

http://localhost:8090
3

Add a Connection

Enter your MCP server URL (Streamable HTTP).

Name: My MCP Server
URL:  http://localhost:3000/mcp
Type: Streamable HTTP
4

Explore

Browse Tools, Prompts, Resources, or open Chat.

# Use Ctrl+K to jump anywhere
# instantly from the keyboard
