---
title: "Dev Tunnels"
description: "Create public webhook endpoints, capture incoming traffic in real time, and replay requests — all from the browser."
weight: 12
---

Turn MCP Explorer into a live webhook control room. Dev Tunnels wraps Microsoft's `devtunnel` CLI to give your local MCP server a public HTTPS callback URL, captures every incoming request as it arrives, and lets you inspect or replay those requests without leaving the app.
