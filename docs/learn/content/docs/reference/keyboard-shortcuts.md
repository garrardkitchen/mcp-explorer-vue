## Command Palette

Press `⌘K` (macOS) or `Ctrl+K` (Windows/Linux) from anywhere in the app to open the command palette.

![Command palette open with fuzzy search and keyboard navigation](/images/screenshots/command-palette.png)
*The command palette provides instant keyboard-first navigation to any page or action. Type to fuzzy-search.*

Type to fuzzy-search and press `Enter` to navigate. Press `Esc` to close.

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Open command palette |
| `Esc` | Close dialogs / command palette |

---

## Chat Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | Insert newline |
| `/` (at start of input) | Open slash commands |
| `Ctrl+Enter` | Send (alternate) |

---

## Slash Commands (Chat)

Type `/` in the chat input to access:

| Command | Description |
|---------|-------------|
| `/clear` | Clear the current conversation |
| `/prompt` | Browse and insert an MCP prompt |
| `/help` | Show all available slash commands |

---

## Tool Invocation

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Run tool / execute prompt |

## Navigation

| Shortcut | Action |
