Prompt Picker
/prompt [search]
Opens a searchable list of all MCP prompt templates available from your connected servers. Select a prompt, fill in any required parameters, and inject the rendered content directly into the chat input.
Example:
/prompt summarize → opens picker filtered to "summarize"
/prompt → opens picker showing all prompts
All macros — full reference
/prompt — Pick & inject an MCP prompt template (supports optional search filter)
/recent — Re-insert one of the last 3 messages you sent
/stats — Show token usage inline (input / output / total this session)
/report — Copy entire conversation as formatted Markdown to clipboard
/system — Open system prompt editor panel
/clear — Clear current conversation (with confirm)
/model — Quick-switch the active AI model
/new — Start a fresh chat session
